## Используемые термины

1. Пользователь — пользователь Системы, обладающий набором Ролей.

2. Роль — понятие, определяющее права Пользователя.

3. Фиксированная Роль – Роль, подразумевающая, что Пользователь обладает только этой Ролью и никакими другими и состоит только в одном Сообществе. (Участник, Наблюдатель)

4. Нефиксированная Роль – Роль, подразумевающая, что Пользователь может обладать этой Ролью одновременно с другими Нефиксированными Ролями и состоять в нескольких Сообществах. (Все Роли помимо указанных в предыдущем пункте)

5. Разработчик — Роль, наделяющая Пользователя правом создания Задач и Туров.

6. Участник — Роль, наделяющая Пользователя правом состоять в Соревновании, принимать участие в доступных Турах.

7. Ученик — Роль, наделяющая Пользователя правом состоять в Классах, принимать участие в доступных Турах.

8. Организатор — Роль, наделяющая Пользователя правом создавать Соревнования/Классы и управлять ими.

9. Наблюдатель — Роль, наделяющая Пользователя правом просматривать результаты Соревнований.

10. Судья — Роль, наделяющая Пользователя правом просматривать Решения и изменять Вердикт.

11. Администратор — Роль, наделяющая Пользователя правом создавать Сообщества и управлять ими.

12. Супервайзер — Роль, наделяющая Пользователя правом создавать других Пользователей и управлять ими.

13. Кабинет — веб\-страница, через которую Пользователь получает доступ к Системе.

14. Файл — загруженный в систему файл (в конвенциональном определении), имеющий уникальный идентификатор.

15. Полигон — тип Файла, являющийся корректной моделью мира TRIK Studio.

16. Условие — тип Файла, являющийся PDF/TXT документом.

17. Упражнение — тип Файла, являющийся корректной программой для TRIK Studio, с недоступной для изменения моделью мира.

18. Решение — тип Файла, являющийся корректной программой для TRIK Studio.

19. Вердикт — результат оценки Решения.

20. Задача — набор Файлов, обязательно состоящий из как минимум одного Полигона, одного Условия, одного Упражнения и одного авторского Решения. Также имеющий обязательное название, опциональное описание, опционально ожидаемые Вердикты для каждого авторского Решения.

21. Тур — набор Задач, имеющий ограничения на время выполнения.

22. Соревнование — набор, состоящий из Участников, Туров и Организатора.

23. Класс — набор, состоящий из Учеников, Туров и Организатора

24. Сообщество — набор Пользователей.

25. Код-доступа (далее – КД) — набор символов.
